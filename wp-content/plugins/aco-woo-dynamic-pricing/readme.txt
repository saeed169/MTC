== HOW TO APPLY BULK DISCOUNTS OR SET DISCOUNT RULES? ==
After activating WooCommerce Dynamic Pricing and Discount Rules plugin, go to the menu Pricing Rules in WordPress left menu bar.

Here you can add new pricing or discount rules by clicking on the 'Add New Rule' button. (Please refer to our video and screenshots below for a better understanding.)
Setting WooCommerce dynamic discounts

In the discount rules form, you can fill following fields:

🔹 <strong>Rule Name</strong> - Name for the discount rule - add any name of your choice

🔹 <strong>Discount Label</strong> - It's the label that will be visible for customers in the Cart, Checkout, and Order Page, which shows how much discount is being applied (It can be something like, 'New Year offer', 'Summer Sales', etc).

🔹 <strong>Priority</strong> - If there are multiple discount rules, you can set priority on which one to be applied first.

🔹 <strong>Product List</strong> - This is the list that you can create by going to WordPress backend >> WooCommerce >> 'Product Lists'. It's the list of products where you want to apply this discount rule. There are further parameters that you can select while creating a 'Product List'.

🔹 <strong>Discount Type</strong> - You can select the 'Discount Type' - that if to apply a discount based on Cart Price or on Product Price, or based on a Fixed price or a Percentage value.

🔹 <strong>Discount Value</strong> - It is this value that will be applied as a discount. In the case of Percentage Type, add the percentage value (Eg: 10 for 10%) and in the case of Fixed Type, add the fixed price (Eg.120).

== Frequently Asked Questions ==
= Is it possible to set multiple discounts? =
Yes, you can set multiple discounts.

= Is it possible to exclude products on SALE? =
Yes, there is the option to exclude products on sale from the discount rules.

= Is it possible to set discounts for selected products? =
Yes, you can create a product list with specific products or can choose taxonomies. 

= Can I set discount rules only for registered customers? =
Yes, there is an option to set discounts only for registered customers.

= Is it possible to exclude selected products from the list? =
Yes, while creating product lists you have the option to exclude selected products from the list 

= If I have multiple rules on the same product, can i set priority for each rule? =
Yes, you can set priority for each rule. Please note cart based rules will have the least priority, will be applied only when a product gets added to the cart.

= How to set BOGO rules for products? =
Just set the discount type as 'Buy X Get X' and set BOGO options (Pro Version).

= Is it possible to gift a product? =
Yes, you can either set a specific product as a gift or can allow users to select the gift from a set of products (Pro Version).

= Can I set discounts for specific user roles? =
Yes, just navigate to the 'Rules and Restrictions' tab and apply user rules (Pro Verison).

= Can I set usage limits for discounts? =
Yes, you can set usage limits from the 'Usage Limits' tab (Pro Version). 

= How to set upcoming sales / discounts? =
You can create discount rules and set desired start / end dates. You can even set multiple start / end dates. In our pro version, you have the option to select weekdays / time. 

= Can I list all the active discount rules on the page? =
Yes, we do provide shortcodes for listing active discounts, you can also add attributes (rule ids) to list specific rules / discounts (Pro Version).

= How can I show the pricing table for quantity-based discounts? =
While setting quantity-based discount rules, you do have the option to show / hide pricing tables. We also provide option to change the pricing table layouts (verticle or horizontal).

= Can I change the pricing table position? =
Yes, the position can be changed from our plugin's settings section, you can also change the pricing table titles / headings from the settings section.

= Can I customize the pricing table view? =
Yes, you can change the border color to match your theme style. We have also provided options for changing table titles, and layouts (horizontal and vertical).

= What happens after my PRO license expires? =
You will be able to use plugin in your site, but however, you won't be able to receive any further updates and priority support.
